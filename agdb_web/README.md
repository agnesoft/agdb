# agdb Documentation

[![Nuxt UI](https://img.shields.io/badge/Made%20with-Nuxt%20UI-00DC82?logo=nuxt&labelColor=020420)](https://ui.nuxt.com)

Documentation site for [agdb](https://github.com/agnesoft/agdb) - Application native database for any use case.

**no query language** | **performance independent of data size** | **10X cost reduction**

- [Main Repository](https://github.com/agnesoft/agdb)
- [Documentation](https://agdb.dev)

## Setup

Make sure to install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
